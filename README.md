# Teical Web Interface

Teical is a centralized marketplace for real estate auctions, designed to simplify the search and analysis of auction properties. This repository contains the frontend application developed with high-performance web technologies.

## Key Features
- **Centralized Listings:** Aggregates real estate auctions from multiple sources in one place.
- **AI-Powered Property Valuation:** Integration with Python-based AI models for accurate market price evaluation and investment analysis.
- **Modern UI/UX:** Fully responsive interface designed in Figma and implemented with Tailwind CSS and Flexbox.

## Tech Stack
- **Framework:** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
